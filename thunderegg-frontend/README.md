# thunderegg-frontend

A VueJs 3.x dApp using Ethers.js

Demo: https://thunder-egg.web.app/ - please use RINKEBY and contact us for staking tokens.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
